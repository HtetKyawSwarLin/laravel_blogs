
# Blogs 

Simple Blogs website with search and security features


## Front End

- Bootstrap

## Back End

- Laravel
- MySQL Database

## Features

- Register | Login
- Create | Read | Update | Delete
- Filter | Search
- Authentication | Authorization


## Installation

```bash
  composer install
  npm install
```
